# An Examination of the Causal Relationship Between Rainfall and Crop Prices: A Focus on Wheat and Corn.

## Description

Group project for CS418 @ UIC (University of Illinois at Chicago) for A.Y. 2024/2025.
Investigating whether analysis of weather data of major corn producers States in the U.S. can lead to informed prediction of future corn prices.
Please refer to the `docs/` folder for more detailed information.

## Reproduce results

To reproduce these results, import project on Google Drive, unzip data.zip and run notebook

## Conclusion

A relation between weather in major corn produces in the U.S. and can be exploited to predict corn prices. Our model exhibts a **R<sup>2<sup>=0.98** and a **Directional accuracy = 0.76**.
